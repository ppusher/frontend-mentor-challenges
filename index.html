About

Frontend Mentor challenges help you improve your coding skills by building realistic projects.

If done correctly, you can learn new things in every challenge, some more so than others. I tried to document everything I learn from each challenge in the README of each challenge. The repo currently holds 50 completed challenges.

But this site showcases a select few challenges, the fun or ones I feel I learned a lot from.
